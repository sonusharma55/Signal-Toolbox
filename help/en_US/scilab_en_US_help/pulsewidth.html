pulsewidth

This function estimate pulse width of real vector x.

Calling Sequence

p=pulsewidth(x)
p=pulsewidth(x, Fs)
p=pulsewidth(x, t)
p=pulsewidth (x, t, 'Polarity', pol)
p=pulsewidth(x, t, 'MidPercentReferenceLevel', N )
p=pulsewidth(x, t, 'Tolerance', M)
p=pulsewidth(x, t,'StateLevels', O)
[p initialcross finalcross nextcross midreference]=pulsewidth(x)
[p initialcross finalcross nextcross midreference]=pulsewidth(x, t)
[p initialcross finalcross nextcross midreference]=pulsewidth(x, t)
[p initialcross finalcross nextcross midreference]=pulsewidth(x, t, 'Polarity', pol)
[p initialcross finalcross nextcross midreference]=pulsewidth(x, t, 'MidPercentReferenceLevel', N )
[p initialcross finalcross nextcross midreference]= pulsewidth(x, t, 'Tolerance', M)
[p initialcross finalcross nextcross midreference]= pulsewidth(x, t,'StateLevels', O)
[p initialcross finalcross nextcross midreference]= pulsewidth(x, t,'StateLevels', O, 'fig', f)

Parameters

x

: real input vector

Fs

: specifies the sample rate, Fs, as a positive scalar, where the first sample instant corresponds to a time of zero.

t

: defiene instant sample time t as vector with same length of x, or specifies the sample rate, t, as a positive scalar.

'Polarity', pol

: pol specify the polarity of the pulse as either 'positive' or 'negative', where the default value is 'positive'

'MidPercentReferenceLevel', N

: specify the mid percent reference leves as a percentage, default value of N is 50.

'Tolerance', M

: define the tolerance value as real scaler value, where default value of M is 2.0.

'StateLevels', O

: O define the lower and upper state levels as two element real vector.

'fig', f

: f specify the logical input value to display figure as one of 'on' or 'off', where the default input in 'off'.

p

: returns a vector of difference between the mid-crossings of the initial transition of each positive-polarity pulse and the next positive-going transition

initialcross

: returns a vector of initial cross values of bilevel waveform transitions x.

finalcross

: returns a vector of final cross values of bilevel waveform transitions x.

nextcross

: returns a vector of next cross values of bilevel waveform transitions x.

midreference

: return mid reference value corrosponding to mid percent reference value.

Examples

x = fscanfMat("macros/pulsedata_x.txt");  //Importing samples of pulse
t = fscanfMat("macros/pulsedata_t.txt");
clf                                       //clearing plot history
p = pulsewidth(x,t,'fig','ON')           // Detremining pulse width and ploting it
