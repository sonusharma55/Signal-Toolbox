zplane

Pole-Zero plot for Discrete time systems

Calling Sequence

zplane(z);
zplane(z,p);

Description

This function gives pole zero plote of discrete time systems

Parameters

z

: vector containing numerator coefficients

p

: vector containing denumerator coefficients

Examples

z = [1 2 3];
p = [4 5 6];
zplane(z,p);
